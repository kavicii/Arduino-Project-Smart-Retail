## Project Objective
1. Create a safe shopping environment
2. Amplify the information of COVID-19
3. Reduce foot traffic in retail shop
4. Enhance the customer's shopping experience

## System Function
1. Sign up membership
2. Update membership record
3. Record visit timestamp of customer
4. Display name, price and location of recently-bought item
5. Record and display reward point
6. Send email to customer automatically

## Equipment List
- Hardware 
```
  Personal computer 

  Arduino Mega 2560 Board 

  I2X bus Passive OLED (0.96 inch, 128x64 dots) 

  Proximity Coupling Device with MFRC522 Contactless Reader 

  RFID tags 

  ESP8266 Wi-Fi module 

  Wi-Fi Router   
```

- Software 
```

  Arduino IDE 

  Python 3.9.4 

  SQLite (Library) 
```
